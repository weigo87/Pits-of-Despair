All Items follow the naming ZB(5 characters)(1 character of definition). For example: Eshpel is ZBeshpeC.bam for the Description BAM. ZBeshpei.bam for the Icon BAM. ZBeshpel.cre for the creature file. ZBeshpel.itm for the item file. ZBeshpeP.png for the portrait. ZBespel.dlg for dialog.

Portrait Icons must be 16x16 pix.
